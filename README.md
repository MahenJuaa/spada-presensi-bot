# SPADA Auto Presensi

## Cara Pakai

1. Buat file `.env` dari `.env.example` dan isi sesuai kredensial kamu.
2. Jalankan dengan `python main.py`
3. Gunakan Railway atau Render untuk hosting otomatis.
